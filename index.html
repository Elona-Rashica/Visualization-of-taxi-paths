Instructions for taxi paths visualization
  1. Upload CSV files with taxi data
  2. Select dates and times for which you want to filter data
  3. Select 'Visualize'
Instructions for the graph
  1. Select 'See graph'
  2. Select dates or leave the default dates
  3. Select 'See graph'
